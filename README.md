# Projeto Biblia Online (FrontEnd)

Projeto de final de curso da Tripleten Brasil. Hospedado na url ......

## Objetivo

Criar um aplicativo FrontEnd para consulta de versículos bíblicos utilizando tecnologias atualizadas.

## Conceitos Chave:

- Permitir que o usuário consulte um capítulo ou versículo indicado da Bíblia.
- Permitir que o usuário consulte um versículo aleatório da Bíblia.
- Permitir que o usuário consulte as instruções corretas para o bom funcionamento da busca.

## Funcionalidades

- Página inicial
- Pesquisar
- Mostrar um versículo aleatório
- Instruções
- Preloader

## Tecnologias

As seguintes tecnologias foram aplicadas neste projeto: JSX, CSS, Hooks do React, API e React Router.
